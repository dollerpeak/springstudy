html구조

html
└ head
   └ meta, title, style, script

   └ content

시멘틱

header, main, footer, section
body 부분을 크게 header, main, footer 로 나눌수 있음
기존 영역을 나누던 div 태그 대신 section 사용

section 태그

nav, aside, article
nav : 주로 링크가 연결되어 있는 메뉴에 사용
article : 내용
aside : nav, article 을 제외한

선택자, selector

tag: 태그명 그대로, h1
class: 점으로 표시, .클래스명
id: #으로 표시, #id명

콤비네이션

(A B: tag, class, id 일 수 있음)
(최종적으로 B를 찾는 방법)
Descendant selector: 자손, A B
Child selector: 자식, A > B
Adjacent Sibling selector: 이웃한, A + B
General Sibling selector: 이웃들, A ~ B

속성을 이용

#ID a[href="a.html"] : ID 손자중에 a태그를 사용하고 값이 href="a.html" 인것만 선택
그외 여러가지가 있는데 많이 사용 안하므로 필요시 마다 찾아서 사용

우선순위

범위가 좁은게 우선순위를 갖는다
id > class > tag
