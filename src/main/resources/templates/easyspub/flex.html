[2023-06-21] flexbox

참고 사이트 : https://developer.mozilla.org/ko/
주요개념
  • 플렉스 컨테이너
  • 플렉스 항목
  • 주축
  • 교차축
플렉스 컨테이너 : flex 설정한 사항이 모두 담긴다. display=flex
주축 : flex-direction 으로 설정, 방향 : row, row-reverse, column, column-reverse
flex-wrap : 컨테이너를 벗어난 값을 안으로 넣는다.
주축 설정 : justify-content, 주축을 기준으로 배치, 플렉스 박스의 배치 방법 결정
교차축 설정 : 한줄인경우=align-items, 여러줄인경우=align-content, 주축을 기준으로 배치
