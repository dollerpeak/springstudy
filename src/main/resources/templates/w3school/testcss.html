설명

기존 html 파일을 좀 더 다양하게 사용가능
html + CSS 구성으로 쓰지 않는 것이 있을까?

적용방식

(inline) html 태그에 style 속성으로 추가
(internal) html head 영역에 style 로 적용해서 해당 페이지 전체에 적용 가능
(external) link 로 외부 CSS파일에 연결 (link rel="stylesheet" href="mystyle.css")

head에 style로 정의 및 사용하는 방법

  • 일반적인 사용

    • html 태그를 직접 지정

    - 선언 : h1 {font-size:10px; color:red}
    - 사용 : h1 태그를 사용할때 마다 선언한 값이 그대로 적용

    id를 사용해서 지정

    - 선언 : #car {font-size:10px; color:red}
    - 사용 h1 id="car" {font-size:10px; color:red}, id를 선택해서 사용

    class를 사용해서 지정

    - 선언 : .car {font-size:10px; color:red}
    - 사용 h1 class="car" {font-size:10px; color:red}, id를 선택해서 사용

    여러개를 선언 할때는 h1, p {font-size:10px; color:red}

    여러개를 사용할때는 id 또는 class = "지정 지정 지정 .."

  • Combinators : 복잡하네 그냥 개념만

    • Descendant

    - 선언 : div p {font-size:10px; color:red}
    - 사용 : div 태그안에 p 태그가 있으면 적용

    Child

    - 선언 : div > p {font-size:10px; color:red}
    - 사용 : div 태그안에 p 태그가 바로 아래 뎁스에 있을때만 중간에 다른 태그가 있으면 적용되지 않는다.

    Adjacent Sibling

    - 선언 : div + p {font-size:10px; color:red}
    - 사용 : div 태그가 끝나고(/div) p 태그가 나타날때

    General Sibling

    - 선언 : div ~ p {font-size:10px; color:red}
    - 사용 : div 태그가 끝나고(/div) 이후 나타나는 모든 p 태그에서 태그가 끝날때(/p) p 태그는 모두 영향을 받는다.

  • Pseudo-classes

    • - 특정상태에 따라를 지정해서 사용

    Anchor

    - 선언 : a:hover {font-size:10px; color:red}
              a:visited {font-size:10px; color:green}
    - 사용 : a 태그 사용 시 방문한 페이지 이면 visited, 마우스가 포커싱되면 hover 적용

    Classes

    - 선언 : a.highlight:hover {font-size:10px; color:red}
    - 사용 : a 태그 사용 시 class="highlight" 사용하면 적용

    div에 hover

    - 선언 : div:hover {font-size:10px; color:red}
    - 사용 : div 태그에 마우스 포커스 되면 적용

  • Attribute

    • - 태그에 특정 속성에 있을때 적용

    태그[속성]

    - 선언 : a[target] {font-size:10px; color:red}
    - 사용 : a 태그에 target 속성이 있는 경우 적용

    태그[속성="값""]

    - 선언 : a[target="_blank"] {font-size:10px; color:red}
    - 사용 : a 태그에 target 속성과 같이 모두 동일할때 적용

  • 그외 무수히 많음, 필요할때 마다 찾아서 하는 걸로
