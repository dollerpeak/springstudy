Spring Menu

/static
- url location정보로 직접 들어올 수 있는 정적페이지 영역
- spring 구동 시 /static/index.html 를 찾는다

/templates
- url location정보로 접근할 수 없는 동적페이지 영역
- 리소스에 접근하려면 resource 탐색영역을 수정해야 하는데 thymeleaf 를 사용하면 반영 됨
- url로 접근하려면 control영역에서 mapping이 필요
- thymeleaf의 디폴트 properties에
spring.thymeleaf.prefix=classpath:/templates/
spring.thymeleaf.suffix=.html
가 적용되어 있어 contorl에서 링크 시 파일명으로만 이동이 가능
- *.html파일이 위치
